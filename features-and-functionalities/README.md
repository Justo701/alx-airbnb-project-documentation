# Airbnb Clone Backend Features

## Core Features
- User Authentication (Register, Login, JWT)
- Property Management (Add, Edit, Delete listings)
- Booking System (Book, Cancel, View bookings)
- Payment Processing (Stripe or PayPal)
- Review & Rating (Users leave reviews)
- Notification System (Emails, Alerts)
- Admin Panel (User and Property management)
