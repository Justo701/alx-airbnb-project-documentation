# User Stories

As a **Guest**, I want to **register an account** so that I can **book properties**.

As a **Guest**, I want to **search for available properties** by **location, dates and number of guests** so that I can **find a place to stay**.

As a **Guest**, I want to **view detailed property information** including **photos, amenities and reviews** so that I can **make an informed booking decision**.

As a **Guest**, I want to **create a booking** after **selecting dates and confirming availability** so that I can **reserve the property**.

As a **Guest / Host**, I want to **make a secure payment** so that the **booking can be confirmed**.

As a **Host**, I want to **list a new property** with **photos, description, price and availability** so that **guests can discover it**.

As a **Host**, I want to **accept or reject booking requests** so that I can **control my calendar**.

As a **Guest**, I want to **leave a review and rating** after my stay so that I can **share my experience**.

As an **Admin**, I want to **manage users and properties** so that I can **moderate the platform**.
